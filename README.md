Install requirements: python, synology-api, python-dotenv

Put variables to .env file.

Example:

```
DOCKER_CONTAINER=some-container-api
FILE_PREFIX=app-name
POSTGRES_USER=postgres
SYNOLOGY_PASSWORD=strongpassword
SYNOLOGY_PATH=/MySharedFolder
SYNOLOGY_PORT=5001
SYNOLOGY_USER=restricteduser
SYNOLOGY_HOST=someaddress.com
```
